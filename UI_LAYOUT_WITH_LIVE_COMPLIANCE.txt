1. PLAY BUTTON (▶)
   → Запускает анимацию
   → Live Compliance начинает расти от 0%
   → Обновляется каждый кадр (60 FPS)

2. PAUSE BUTTON (⏸)
   → Останавливает анимацию
   → Live Compliance замирает на текущем значении

3. TIME SLIDER (═══)
   → Перемотка на любой момент времени
   → Live Compliance пересчитывается для этого момента
   → Показывает compliance до выбранной точки

4. SPEED SELECTOR (1×, 2×, 4×...)
   → Ускоряет воспроизведение
   → Live Compliance обновляется быстрее
   → Расчеты остаются точными

5. RADIO BUTTON (◉)
   → Выбор траектории для демонстрации
   → Live Compliance сбрасывается при смене

╔═══════════════════════════════════════════════════════════════════════════════════════════════╗
║                                    ЧТО СЧИТАЕТСЯ                                               ║
╚═══════════════════════════════════════════════════════════════════════════════════════════════╝

Live Compliance = (Количество compliant точек / Всего пройденных точек) × 100%

Точка COMPLIANT если:
    ✅ Deviation <= Allowed Deviation (расчет C++)
    AND
    ✅ Speed <= Allowed Speed (расчет C++)
